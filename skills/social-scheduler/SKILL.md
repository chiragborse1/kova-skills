# Social Post Scheduler

When user provides a topic or idea, write platform-optimized versions for:
- **Twitter/X**: Max 280 characters, punchy, use 1-2 hashtags
- **LinkedIn**: Professional tone, 2-3 paragraphs, value-focused
- **Telegram/Discord**: Conversational, can be longer, use formatting

Label each version clearly.
